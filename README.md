# concurrent

Tiny, practical Go concurrency helpers:
- **Worker Pool** – simple fan-out/fan-in with `context` cancellation.
- **Pipeline** – composable channel stages (`Map`, `Filter`, `Batch`) with clean close.
- **MapConcurrent** – bounded parallelism over a slice.

## Install

```bash
go get github.com/logimos/concurrent
```

Documwntation can be found [here](https://logimos.github.io/concurrent/)
